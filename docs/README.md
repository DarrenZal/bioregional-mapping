# docs/

Canon-bearing structure for bioregional-mapping.

## Layout

- `foundations/` — local foundation docs (zero admitted at minimal-skeleton phase; bioregional-coordination + Spore foundation docs cited at upstream-reference layer)
- `research/canon-decisions/` — ADRs (0001 establishes repo identity + bioregional-coordination relationship)
- `research/connections/` — bridge notes for cross-repo coordination (BKC + SalishOakSeeds peer-relationship cross-citation, comparative intake of external work, RC LHC Phase 2 sources, Tijn-thread tooling-map, BioKulture blue paper, etc.)
- `workshops/` — draft workshop packets, rehearsals, and facilitation planning artifacts (empty at minimal skeleton)

## Conventions

Inherits Spore + bioregional-coordination conventions adapted for `bioregional-mapping` namespace:

- doc_id: `bioregional-mapping.<slug>`
- doc_kind: `foundation | decision-record | connection | positioning | pattern | spec | architecture | research | planning | vision`
- status: `draft | active | superseded | archived | deprecated`
- Cross-repo citations by ADR number (`spore:ADR-NNNN-<slug>`, `bioregional-coordination:ADR-NNNN-<slug>`), not commit SHA

See `../CLAUDE.md` for full method discipline.
