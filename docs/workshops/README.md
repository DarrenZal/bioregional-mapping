# Workshops

Draft workshop packets, rehearsals, and facilitation planning artifacts for bioregional mapping practice.

**Current state (2026-05-04)**: empty (minimal-skeleton phase). Populate when a real mapping workshop earns canonical articulation (e.g., abstracted methodology after Victoria workshop runs in June–July 2026 via SalishOakSeeds).

## When to admit a workshop packet

Workshop packets admit here when:

- The workshop methodology is being abstracted across hubs (not Victoria-specific — that lives in SalishOakSeeds; not Boulder-specific — that will live in a future Boulder hub repo)
- The packet captures reusable scaffolding (timing, layer rubric, facilitation moves, documentation templates)
- It is NOT a content artifact (specific maps, specific outputs, specific community data — those live in instance-deployments)

## Reference

bioregional-coordination's [`docs/workshops/victoria-workshop-simulator-v0/`](https://github.com/DarrenZal/bioregional-coordination/tree/main/docs/workshops/victoria-workshop-simulator-v0) is the structural template — facilitation packet, role cards, diagnostic worksheet, route board, viability band board. When admitting a packet here, use that structure.
