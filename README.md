# bioregional-mapping

A canon-bearing repository for **bioregional mapping practice** — multi-layer mapping-as-coordination across relational, cultural, ecological, economic, governance, and geospatial layers.

This repo specializes [bioregional-coordination](https://github.com/DarrenZal/bioregional-coordination) at the mapping sub-domain, and composes (transitively, via bioregional-coordination) with [Spore](https://github.com/DarrenZal/spore) coordination grammar at the upstream-reference layer. It does not redefine grammar; it instantiates upstream primitives at the mapping practice and grows local doctrine where mapping-specific concerns require it.

## What this repo owns

**Cite this repo for:** the multi-layer mapping framing (six layers and how layer multiplicity is canonical even when layer naming is provisional); how Spore primitives + bioregional-coordination specializations apply specifically at the mapping practice (e.g., layer-as-membrane, Story-of-Place as Field manifestation, workshop-as-Joint-Commitment, Field Guide as Reproduction artifact); the bridge-note index of external practitioner work and tooling for bioregional mapping (RC Phase 2 Program Guide, Tijn Tjoelker thread tooling-map, BioKulture blue paper, Joe Brewer webinar, etc.).

**Don't cite this repo for:** coordination-grammar canon (Spore owns); bioregional-scope specialization (bioregional-coordination owns); operational instance machinery, mapping data, monitoring records, pilot specifics, or the seven-layer mapping-to-finance stack (BKC + instance-deployments own); RC Field Guide content (Regenerate Cascadia owns; this repo links).

For the full inter-canon role-pinning across Spore + bioregional-coordination + bioregional-mapping + BKC + IC + PM, see [Spore README §Canon scope and inter-canon roles](https://github.com/DarrenZal/spore#canon-scope-and-inter-canon-roles).
